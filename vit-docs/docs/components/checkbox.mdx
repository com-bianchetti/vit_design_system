# Checkbox

Checkboxes are interactive controls that allow users to select one or multiple options from a set. They provide clear visual feedback and are ideal for enabling/disabling features, accepting terms, or selecting multiple items.

<FlutterPreview story="VitCheckbox" />

## Import

```dart
import 'package:vit_design_system/vit_design_system.dart';
```

## Key Features

- **Standalone or List Mode**: Render as a simple checkbox or as part of a styled list item with title and subtitle
- **Tristate Support**: Optional indeterminate state for parent-child checkbox relationships
- **Flexible Positioning**: Place checkbox on left or right side of list items
- **Form Integration**: Seamless integration with `VitForm` for data collection and validation
- **Visual Density Control**: Adjust spacing with compact, standard, or comfortable modes
- **Skeleton Loading**: Built-in loading state with shimmer effect
- **Accessibility**: Full semantic label and hint support for screen readers
- **Customizable Styling**: Control colors, borders, and backgrounds through theme

## Basic Usage

### Simple Checkbox

A minimal checkbox without any labels, perfect for custom layouts or inline controls.

```dart
VitCheckbox(
  id: 'agree',
  title: 'I agree to the terms',
  value: false,
)
```

### Checkbox with Subtitle

Add a subtitle to provide additional context or explanation for the checkbox option.

```dart
VitCheckbox(
  id: 'marketing',
  title: 'Marketing Emails',
  subtitle: 'Receive promotional offers',
  value: false,
)
```

### Controlled Checkbox

Use controlled state to manage checkbox value externally and respond to changes.

```dart
class MyCheckbox extends StatefulWidget {
  @override
  _MyCheckboxState createState() => _MyCheckboxState();
}

class _MyCheckboxState extends State<MyCheckbox> {
  bool _checked = false;

  @override
  Widget build(BuildContext context) {
    return VitCheckbox(
      id: 'option',
      title: 'Enable Feature',
      value: _checked,
      onChanged: (value) {
        setState(() => _checked = value);
      },
    );
  }
}
```

## Advanced Features

### Checkbox Positioning

Control whether the checkbox appears on the left or right side of list items.

```dart
VitCheckbox(
  id: 'left_check',
  title: 'Left Checkbox',
  subtitle: 'Checkbox on the left',
  checkboxPosition: VitCheckboxPosition.left,
  value: false,
)
```

### Tristate Checkbox

Use tristate mode for parent-child relationships where the parent can be checked, unchecked, or indeterminate.

```dart
VitCheckbox(
  id: 'parent',
  title: 'Select All',
  value: null,
  tristate: true,
  onChanged: (value) {
  },
)
```

### With Icon

Add an icon to provide visual context for the checkbox option.

```dart
VitCheckbox(
  id: 'notifications',
  title: 'Push Notifications',
  icon: Icon(Icons.notifications),
  value: true,
)
```

### Visual Density

Adjust the height and spacing of list-style checkboxes to match your design requirements.

```dart
VitCheckbox(
  id: 'compact',
  title: 'Compact Option',
  visualDensity: VisualDensity.compact,
  value: false,
)
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `id` | `String` | Unique identifier |
| `title` | `String` | Main label |
| `subtitle` | `String?` | Secondary description |
| `value` | `bool` | Checked state |
| `onChanged` | `ValueChanged<bool>?` | Change callback |
| `enabled` | `bool` | Enable/disable |

## Form Integration

Checkboxes integrate seamlessly with `VitForm` to collect and validate user selections. When used with a form, checkbox values are automatically included in the form data map using their `id` as the key.

The form integration provides:
- **Automatic Data Collection**: Values are automatically saved to form data
- **Validation Support**: Add validators to ensure required checkboxes are selected
- **Error Display**: Validation errors appear below the checkbox
- **Multi-page Support**: Use checkboxes across multiple form pages

```dart
VitForm(
  pages: [
    VitFormPage(
      children: [
        VitCheckbox(
          id: 'terms',
          title: 'I accept the terms',
          subtitle: 'Required to continue',
          validator: (value) {
            if (value == null || !value) {
              return 'You must accept the terms';
            }
            return null;
          },
        ),
        VitCheckbox(
          id: 'newsletter',
          title: 'Subscribe to newsletter',
        ),
      ],
    ),
  ],
  onComplete: (data) {
    print('Terms: ${data['terms']}');
    print('Newsletter: ${data['newsletter']}');
  },
)
```
